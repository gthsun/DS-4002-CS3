# CS2-DS4002
This repository contains the study, deliverables, data, and reference materials necessary to complete the Spotify Case Study.
## Important Information
The two attached PDFs contain a document with the case study's prompt and deliverables as well as a rubric that sets you up for success and tells you what is required of you. 
## Data
The data to be used is located in the "Data" folder and includes the original dataset as well as an example of the final, usable dataset just for the years of 1980 and 1981. 
## Materials
The "Materials" folder contains contexual resources, documentation, and research papers that are relevant to this case study. You are highly encouraged to reference them throughout this project to better understand technologies and data you will be working with.
